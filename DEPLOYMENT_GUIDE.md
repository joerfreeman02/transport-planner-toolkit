# GitHub Deployment Guide — Foundation 1.6

1. Extract the ZIP.
2. Open the extracted `transport-planner-toolkit-foundation-1.6` folder.
3. Open the GitHub repository `joerfreeman02/transport-planner-toolkit` on the Code tab and main branch.
4. Choose **Add file → Upload files**.
5. Drag every item from inside the extracted folder into GitHub. Do not upload the ZIP or outer folder.
6. GitHub automatically records same-path files as modified; it normally does not ask a separate replacement question.
7. Commit with: `Establish canonical toolkit foundation 1.6`
8. Open Actions and wait for `pages build and deployment` to show a green tick.
9. Open: `https://joerfreeman02.github.io/transport-planner-toolkit/?build=TPT-FOUNDATION-160-20260714`
10. Confirm the home page and all three working modules open.
