# Deployment Guide — Sprint 2B

1. Extract the ZIP.
2. Open the `transport-planner-toolkit` repository on GitHub.
3. Confirm **Code** and branch **main**.
4. Select **Add file → Upload files**.
5. Drag everything from inside the extracted folder into GitHub.
6. Commit with: `Deploy Sprint 2B assisted rail research`.
7. Open **Actions** and wait for the Pages deployment green tick.
8. Open: `https://joerfreeman02.github.io/transport-planner-toolkit/?build=TPT-S2B-20260714`.
9. Home must show `Sprint 2B Assisted Research · TPT-S2B-20260714`.
