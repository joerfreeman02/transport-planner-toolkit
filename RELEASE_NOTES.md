# Sprint 2 Review — Release Notes

## Implemented
- All nearby station discovery within an editable radius
- National Rail / Underground / Overground / DLR / tram separation
- Walking and cycling routing for each station
- Map markers and selectable route display
- Inclusion/exclusion controls
- Editable operator, frequency, destination, cycle parking and step-free fields
- CSV, copy-table and draft wording outputs
- Local browser saving and diagnostics

## Important limitation
Live authoritative service data is not automatically populated in this review build because the relevant National Rail/operator/TfL sources do not provide a universal free browser API suitable for direct static-site use. The fields are therefore editable and require verification before submission.
