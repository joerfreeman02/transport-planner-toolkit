# Technical Director Decision — 16 July 2026

Sprint 4 — Bus Foundation is approved for implementation.

## Governing amendments incorporated

1. Shared contracts, Bus architecture and frozen-module regression are delivered within Sprint 4; Bus is not delayed behind a standalone readiness project.
2. Phase names are: Phase 1 Core Platform; Phase 2 Expansion; Phase 3 Integrated Reporting; Phase 4 Enterprise / Commercialisation.
3. Dashboard and STATS19 receive controlled identities DASH-1.0.0 and STATS-1.0.0 at the next release without functional change.
4. Report Builder depends upon the Project Data Model, not a full Project Manager UI.
5. Significant architectural decisions use sequential ADRs beginning ADR-001.
6. The Toolkit is a commercial-grade internal platform. AI capability is described as AI-assisted drafting and evidence workflows.

Frozen baselines remain AA-1.2.0, RAIL-4.4.0, Dashboard and STATS19. No other scope change is authorised.
